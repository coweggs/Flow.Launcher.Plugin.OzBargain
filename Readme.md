# Flow Launcher OzBargain Plugin

A plugin for the [Flow launcher](https://github.com/Flow-Launcher/Flow.Launcher) that shows [OzBargain](https://www.ozbargain.com.au/)'s feed.

### Installation
    pm install https://github.com/coweggs/Flow.Launcher.Plugin.OzBargain/releases/download/v1.0.0/Flow.Launcher.Plugin.OzBargain.zip

### Usage
    oz <url>
Once a feed has been fetched, interact with any item to open it in your default browser. Title, and Expiry Date, as well as an Image are shown in Flow Launcher.

If no `<url>` is specified:
1. New Deals: Autofill new deals URL.
2. Freebies: Autofill freebies URL.
3. Popular Deals: Autofill popular deals URL.
4. Refresh: Refresh plugin cache (high usage causes rate limits)
